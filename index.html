LT_HOT-2-两数相加

Posted on 2021-07-04
题目

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

个人思路

这道题一开始认为的易错点就是进位问题，思路就是顺着链表加就完事

注意点

  1. 使用while和链表时候， 节点要移起来（老是忘掉）
  2. 在用虚拟投头节点的时候，再找一个引用去表示它，不然头结点跑掉了
  3. 没有想到在某一个链表循环结束时候，可能另一个链表会进位
  4. 没有想到会一直进位到最后，最后还需判断进位是否存在 需要补个1

  • 初始思路

  • 最帅解法

    原来这个思路是看到了题目里有一个方法 ListNode(int val, ListNode next) { this.val = val; this.next = next; } 很巧妙 用到了递归

LT_HOT_1_两数之和

Posted on 2021-07-04
题目

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

个人思路

这道题一开始想到的就是使用map，但是一直在想k和v到底该放什么 ，如果有重复了元素的话 那么肯定不能放在k上啊 ，但是放在v上又想不出来

想了半天看了下题解，题目表示只有一个有效答案，说明就是target如果是偶数的话，最多有2个相同的元素加起来符合taget

首先去判断map中有没有符合的元素，有的话就直接取map内的和map外 数组内的元素拼接，没有的话扔进去map，避免了判断3，3这种情况
